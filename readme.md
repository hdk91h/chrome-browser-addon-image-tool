# Chrome Browser addon image size & type Tool

Link to the chrome browser app: [Microsoft Edge Store](
https://microsoftedge.microsoft.com/addons/detail/inedchcjifdbdjolkolfhkbbigjbpikh)

-> Your images, in any format, will be transferred to your desired format and, following quality standards, the file will be compressed, reducing its file storage size.
-> Everything works seamlessly in your Chromium browser (Google Chrome, Opera, Microsoft Edge, ...)
-> Privacy is ensured as no data is collected; the app operates autonomously within your browser.

The application utilizes methods that are still considered experimental in current browsers.
